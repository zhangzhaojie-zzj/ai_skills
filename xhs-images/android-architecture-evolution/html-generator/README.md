## 🚀 使用方法

### 方法一：浏览器直接导出（推荐）

1. **打开文件**
   ```bash
   open xhs-images/android-architecture-evolution/html-generator/index.html
   ```
   或用浏览器直接打开该文件

2. **导出图片**
   - 点击右下角的 **"📸 导出图片"** 按钮
   - 系统会自动下载 5 张 PNG 图片
   - 图片尺寸：1800×2400 像素（高清）

### 方法二：打印为 PDF

1. 点击 **"🖨️ 打印/保存PDF"** 按钮
2. 在打印对话框中选择 "存储为 PDF"
3. 每张卡片会自动分页

### 方法三：手动截图

1. 在浏览器中打开文件
2. 使用系统截图工具截取每张卡片
3. 推荐分辨率：900×1200 或更高

## ⚠️ 注意事项

1. **浏览器兼容性**: 推荐使用 Chrome、Safari 或 Edge
2. **导出时间**: 点击导出后请等待几秒钟，5张图片会依次下载
3. **图片命名**: 导出的图片会自动命名为 `android-architecture-01.png` 至 `android-architecture-05.png`
4. **网络依赖**: 导出功能需要联网加载 html2canvas 库

## 🔧 离线使用

如需离线使用导出功能：

1. 下载 html2canvas 库到本地
2. 修改 index.html 中的脚本引用：
   ```html

   <script src="./html2canvas.min.js"></script>
   ```
